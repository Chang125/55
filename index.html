操作解說(Netlify)

1.從Twine那邊按Publish to file
2.下載下來後要將你的檔案名稱改成index.html並放進資料夾裡(資料夾名稱不限制)
3.註冊並登入Netlify(新手通常用email註冊即可)
4.到主頁會看到Sites，將資料夾拖曳進去就完成了(見下方圖片)



操作解說(Github)

1.從Twine那邊按Publish to file
2.下載下來後要將你的檔案名稱改成index.html並放進資料夾裡(資料夾名稱不限制)
3.註冊並登入Githuub，登入後按右上角第一個選項(new repository)


4.Repository name的名字隨便取，並按下Create repository
5.按下藍色有底線的uploading an existing file就可以加入你的檔案了
6.在Setting找到Pages，找到source並把branch改成main，上面就會跑出網址。
